The BLACC Community Platform serves as a central hub for black developers to connect, share knowledge, and support each other in tech. The platform creates a welcoming environment where members can engage with content and participate in the community.

## Overview

The community hub provides a unified landing page that showcases what BLACC is all about and provides immediate access to join the Discord community.

## User experience

When you visit the BLACC platform, you're greeted with a clean, focused interface that immediately communicates the mission and values of the community.

<Steps>
  <Step title="Landing page">
    The home page features a full-screen jumbotron with the community tagline: "A community exemplifying black excellence and pushing the needle within tech"
  </Step>
  
  <Step title="Join the community">
    Click the "Join our community" button to be redirected to the BLACC Discord server at `https://discord.gg/9Bh3TZ2`
  </Step>
  
  <Step title="Live Discord widget">
    Interact with the Discord widget in the bottom-right corner to see who's currently online and active in the community
  </Step>
</Steps>

## Discord integration

The platform features an embedded Discord widget that provides real-time visibility into community activity.

<CodeGroup>
```jsx Widget implementation
<Widget open={openWidget}>
  <Click onClick={() => setOpenWidget(!openWidget)} />
  <iframe
    src="https://discord.com/widget?id=769610692297752586&theme=dark"
    width="100%"
    height="100%"
    allowtransparency="true"
    frameBorder="0"
    sandbox="allow-popups allow-popups-to-escape-sandbox allow-same-origin allow-scripts"
  ></iframe>
</Widget>
```

```jsx State management
const [openWidget, setOpenWidget] = useState(false);

return (
  <Widget open={openWidget}>
    <Click onClick={() => setOpenWidget(!openWidget)} />
    {/* Discord iframe */}
  </Widget>
);
```
</CodeGroup>

The widget can be expanded or collapsed by clicking on it, providing a non-intrusive way to check community activity.

<Note>
  The Discord widget shows real-time member status and allows quick access to the server without leaving the platform.
</Note>

## Navigation

The platform uses a sticky header that provides easy navigation across all features:

- **Home**: Return to the community landing page
- **Live Events**: Access live and past coding sessions and podcasts
- **Contribute**: Link to the BLACC GitHub organization
- **Support**: Buy Me a Coffee link for community support

## Responsive design
