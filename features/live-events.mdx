BLACC hosts regular live events including coding sessions, technical discussions, and podcasts. The live events feature provides a centralized platform to watch current streams and browse past recordings.

## Overview

The live events page is accessible at `/live` and displays both upcoming/live events and a searchable archive of past sessions. All streams are broadcasted simultaneously to multiple platforms including Twitch, YouTube, and the BLACC platform itself.

## Event card component

The Event component provides a consistent UI for displaying both live and past events:

<CodeGroup>
```jsx Event structure
export default function Event({ img, title, date, url, live }) {
  return (
    <Container>
      <Img>
        <Image src={img} layout='fill' objectFit='cover' />
      </Img>
      <Details>
        <Metadata>
          <Title>{title}</Title>
          {url && <Date>{date}</Date>}
        </Metadata>
        {!url ? (
          <CTA>{date}</CTA>
        ) : !live ? (
          <CTA href={url} target='_blank'>
            VIEW RECORDING
          </CTA>
        ) : (
          <CTA href={url} target='_blank'>
            WATCH LIVE
          </CTA>
        )}
      </Details>
    </Container>
  );
}
```

```jsx Props
{
  img: string,        // Path to event thumbnail
  title: string,      // Event title
  date: string,       // Event date
  url: string,        // Link to stream or recording
  live: boolean       // Whether event is currently live
}
```
</CodeGroup>

The component intelligently renders different CTAs based on the event state:
- **Live event**: "WATCH LIVE" button
- **Past event**: "VIEW RECORDING" button  
- **Scheduled event**: Displays date only

## Page layout

The live events page uses a responsive grid layout:

```jsx Page structure
export default function Live() {
  const container = useRef(null);
  const [height, setHeight] = useState(0);
  const [width, setWidth] = useState(0);

  useEffect(() => {
    if (container?.current) {
      setHeight(container.current.offsetHeight);
      setWidth(container.current.offsetWidth);
    }
  }, [container]);

  return (
    <Container ref={container}>
      {!width || !height ? (
        <p>Loading</p>
      ) : (
        <Wrapper>
          <LiveEvents />
          <PastEvents />
        </Wrapper>
      )}
    </Container>
  );
}
```

<Note>
  The page measures container dimensions on load to ensure the embedded video player renders at the optimal size.
</Note>
