## Welcome to BLACC

Get up and running with the BLACC Community Platform and submit your first video in just a few steps.

<Steps>
  <Step title="Join the Discord community">
    BLACC is a community-first platform. Start by joining our Discord server where you'll connect with other black developers and stay updated on live events.
    
    1. Visit the [BLACC homepage](/)
    2. Click the **Join our community** button
    3. Complete the Discord invitation process
    4. Introduce yourself in the welcome channel

    <Tip>
      Your Discord username will be used when you submit content to the platform, so make sure it represents you well.
    </Tip>
  </Step>

  <Step title="Explore the platform">
    Navigate through the main sections of the BLACC platform to understand what's available.

    ### Homepage (/)
    The landing page showcases the BLACC mission and provides quick access to the Discord community widget in the bottom-right corner.

    ### Live Events
    View upcoming and past community events (see [Live Events](/features/live-events)), including:
    - Live coding sessions
    - Tech talks and workshops
    - Community podcasts

    <Note>
      Live events are streamed on Twitch. Check the Live Events page regularly for upcoming sessions you can join.
    </Note>

    ### Video Upload
    Submit your own coding videos to be featured on the BLACC YouTube channel (see [Video Submissions](/features/video-submissions)). This is where you'll share your knowledge with the community.
  </Step>

  <Step title="Submit your first video">
    Ready to contribute? Let's walk through uploading your first video submission.

    <Note>
      Videos submitted through the platform are reviewed by the BLACC team before being published to the official YouTube channel.
    </Note>

    Navigate to the upload page (learn more in [Video Submissions](/features/video-submissions)) and follow these steps:

    ### Upload your video file

    Click the upload area and select your video file. Supported formats include MP4, MOV, AVI, and other common video formats.

    ```javascript
    // The platform validates file type on upload
    if (!target.files[0]?.type.startsWith('video')) return;
    ```

    <Warning>
      Make sure your video file is ready before starting the upload. You'll need to provide all metadata in the same session.
    </Warning>

    ### Fill in video details

    Provide the following information:

    <CodeGroup>
    ```text Discord Username
    Enter your Discord username exactly as it appears in the community.
    Example: DevName#1234
    ```

    ```text Video Title
    Create a clear, descriptive title for your video.
    Example: "Building with the Twitch API and GitHub Actions"
    ```

    ```text Description
    Write a detailed description of your video content.
    Include what viewers will learn and any prerequisites.
    ```
    </CodeGroup>

    ### Select relevant tags

    Choose one or more tags that describe your video content. Available tags include:

    <CardGroup cols={3}>
      <Card title="Languages" icon="code">
        - JavaScript
        - Python
        - Java
        - C++, C#
        - HTML/CSS
        - Objective-C
      </Card>
      <Card title="Frameworks" icon="layer-group">
        - React
        - Angular
        - Vue
        - Node.js
        - Express.js
        - Flutter
      </Card>
      <Card title="Topics" icon="lightbulb">
        - Cybersecurity
        - Databases
        - CI/CD
        - GraphQL
        - REST APIs
        - IoT
      </Card>
    </CardGroup>

    <Tip>
      You can select multiple tags. Choose at least one to help others discover your content.
    </Tip>

    ### Submit and track progress

    1. Click the **UPLOAD SUBMISSION** button
    2. Watch the progress bar as your video uploads to Firebase Storage
    3. Wait for the success confirmation (green checkmark)

    ```javascript
    // Upload progress is tracked in real-time
    const progress = (snapshot.bytesTransferred / snapshot.totalBytes) * 100;
    ```

    Your video metadata is stored in Firestore and linked to the uploaded file:

    ```javascript
    firestore.collection('youtubeSubmissions').add({
      title,
      storageId,
      description,
      tags: chosenTags,
      video: downloadURL,
      createdAt: firebase.firestore.FieldValue.serverTimestamp()
    })
    ```
  </Step>
</Steps>
